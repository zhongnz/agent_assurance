# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The canonical text — including standards for participation, scope, and enforcement framework — is available at the link above.

## Scope

The Contributor Covenant applies to all project spaces: GitHub Issues, pull requests, code review comments, project Discussions, and any other venue where individuals are representing the project or its community. It also applies when an individual is officially representing the project in public spaces.

## Expected behaviour

Constructive engagement with the methodology and its artifacts; respect for differing viewpoints and experiences; willingness to give and gracefully accept feedback; focus on what is best for the work and the community.

## Unacceptable behaviour

Harassment, personal attacks, dismissive engagement, and any conduct that the canonical Contributor Covenant text identifies as inconsistent with a respectful community.

## Reporting

Concerns about conduct that may violate this code may be raised via the contact mechanism on the maintainer's GitHub profile ([@zhongnz](https://github.com/zhongnz)) or via the channel documented in `SECURITY.md` for sensitive disclosures.

All reports will be reviewed and investigated promptly and fairly. The reporter's identity is treated as confidential to the extent possible while still allowing the maintainer to act on the report.

## Enforcement

The maintainer is responsible for clarifying and enforcing these standards and will take appropriate corrective action in response to behaviour deemed inappropriate, threatening, offensive, or harmful. This may include warning the responsible party, removing the content in question, or, where necessary, restricting the party's participation in the project's spaces.

## Acknowledgement

The Contributor Covenant is a widely-adopted code of conduct for open-source projects. The canonical version 2.1 text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>. The full text governs in any case of ambiguity with the summary above.

## A note on the project's current scale

The project is solo-maintained. Formal enforcement procedures — a conduct committee, an appeals process, structured incident response — are downstream of engagement scale. The maintainer is the responsible party for now and welcomes feedback on how this scales.
